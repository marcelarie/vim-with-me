# vim-with-me
Vim clone with JavaScript.  

![Vim Logo](https://download.logo.wine/logo/Vim_(text_editor)/Vim_(text_editor)-Logo.wine.png)


# Phase 1  

-[x] Text area.  
-[] Left number column.  
-[x] Color palette.  

# Phase 2   

-[] Carret move.  
-[] Save files.   
-[] Normal mode.  
-[] Visual mode.  
-[] Insert mode.   
-[] Move with h j k l.   
-[] Esc to exit to normal mode.    
-[] x to delete a character.     
-[] dd to delete a line.    
-[] yy to copy a line.    
-[] u to go undo.     
-[] Cntrl + R to redo.   

# Phase 3    

-[] : to open terminal mode.     
-[] :%s/word/word-to-replace/g command.    
-[] syntax highlighting for JS.  
-[] Relative numbers.  
-[] Airline.   
-[] NerdTree.    
-[] Split screen.   
-[] .vimrc    

# Phase 4    

-[] Colorschemes.   
-[] Vimtutor.
